# Professional Profile Specification

**Draft**

A standard for professional details.

This specification is a structured yml file containing all professional
details of individual. Professional details such as experience, education,
skills etc are part of this specification.

## Spec

**Personal details**

- `first_name`
- `preferred_first_name`
- `middle_name` 
- `last_name` 

**Contact information**

- `location`
- `country_name`
- `country_code`
- `phone_number`
- `email`
- `linkedin`
- `github`
- `website`

**Experience**
- company\_name
- location
- role
- duration
- contributions

**education**

- name
- location
- duration
- degree\_name
- cgpa
- achievements
