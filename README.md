# Professional Profile Specification

**Note: Work in progress**

A standard for professional details.

This specification is a structured yml file containing all professional details
of individual. Professional details such as experience, education, skills etc
are part of this specification.

## Usage

- Apply to jobs faster
- Register and create profile on job boards faster
- Easy discovery of profiles
- and more

## Specification

See [`schema.py`](./schema.py) for the current specification.

## An example PPS

```yml
status: In progress
```
